# Personal Finance Planner

Приложение для простого и эффективного управления личными финансами.

## Problem
Пользователи испытывают трудности в планировании бюджета и управлении личными финансами из-за недостатка информации и сложностей в использовании существующих приложений. Существующие решения часто предлагают ограниченные возможности настройки и не удовлетворяют потребности пользователей.

## Solution
Создание простого и интуитивно понятного приложения для планирования бюджета, которое предоставляет пользователям возможность легко отслеживать свои расходы, устанавливать финансовые цели и получать рекомендации на основе их финансовых привычек.
