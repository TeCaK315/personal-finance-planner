# Personal Finance Planner

Приложение для простого и эффективного управления личными финансами.

## Problem
Пользователи сталкиваются с трудностями в планировании бюджета и управлении личными финансами из-за недостатка информации и сложностей в использовании существующих приложений, что приводит к неэффективному управлению своими финансами.

## Solution
Создание интуитивно понятного приложения для финансового планирования, которое поможет пользователям управлять своим бюджетом, предоставляя им все необходимые инструменты и информацию в одном месте, с возможностью настройки под индивидуальные потребности.
